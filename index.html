História dos computadores

A história do computador é uma jornada de evolução desde máquinas mecânicas complexas até os dispositivos eletrônicos compactos e poderosos que usamos hoje. A trajetória inclui o conceito de computador programável de Charles Babbage, a invenção do ENIAC (primeiro computador eletrônico), o desenvolvimento de transistores e circuitos integrados, e a criação dos microprocessadores que levaram aos computadores pessoais.

Ferramentas utilizadas

O Visual Studio Code é um editor de código-fonte desenvolvido pela Microsoft para Windows, Linux e macOS. Ele inclui suporte para depuração, controle de versionamento Git incorporado, realce de sintaxe, complementação inteligente de código, snippets e refatoração de código.

Clients

Os clients podem ser compreendidos como as aplicações que solicitam recursos aos servers. Trazendo para exemplos práticos que podem facilitar o entendimento, podem ser aplicativos de celulares, programas de computador e sites na web. Uma analogia que pode ser aplicada, é o exemplo de relação entre um garçom e um cliente em um restaurante, onde o garçom é o server (servidor, aquele que irá atender os pedidos do cliente) e o client é propriamente o cliente que fará os pedidos ao garçom.

Enquanto as aplicações client necessitam e utilizam uma quantidade menor de recursos, as aplicações server costumam consumir e necessitar de estruturas mais robustas para suportar as diversas requisições dos clientes.

Servers

Os servers podem ser compreendidos como as aplicações que processam as solicitações dos clientes, aplicam as regras de negócio necessárias e respondem aos clientes entregando os recursos solicitados. A estrutura por trás dos servers costumam ser mais complexas e invisíveis aos usuários, apenas uma camada mais simples é apresentada e determina os recursos disponíveis e as condições necessárias para serem acessados.

Estrutura do HTML

A estrutura do HTML é composta por três partes principais.

  • Tag html

    A tag html envolve todo o documento HTML, indicando ao navegador que o conteúdo deve ser interpretado como uma página web.

  • Tag head

    A tag é carregada antes do corpo da página e serve para definir configurações, metadados e recursos, como o título da aba, links para arquivos CSS, scripts e informações para buscadores.

  • Tag body

    A tag envolve todos os elementos visíveis ao usuário, como textos, imagens, botões e formulários.

Tags

Algumas tags foram apresentadas no módulo 1, veja um pouco mais sobre elas:

  • tags h1 ao h6

    As tags h1 ao h6 são tags de título e possuem 6 níveis, ao utilizar uma dessas tags o navegador entende que é uma tag de título e aplica comportamentos padrões como o tamanho de acordo com cada nível de título e em negrito.
    Exemplos:

    Título h1

    Título h2

    Título h3

    Título h4

    Título h5
    Título h6

  • tag p

    A tag p é utilizada para delimitar um parágrafo na estrutura do html. Exemplos:

    Este é o parágrafo 1

    Este é o parágrafo 2

  • tag mark

    A tag mark é utilizada para aplicar o comportamento de marcação de texto, por exemplo, o texto marcado na cor amarela

  • tag small

    A tag small serve para indicar que um trecho de texto deve ser exibido em tamanho menor que o texto normal, geralmente para observações, comentários, avisos ou notas de rodapé.
    Exemplo:

    Este é um parágrafo normal. Este é um texto menor dentro do parágrafo.

  • tag i

    A tag i serve para deixar o texto em itálico.
    Exemplo:

    O livro Dom Quixote é um clássico.

  • tag u

    A tag u serve para deixar o texto sublinhado.
    Exemplo:

    Este é um texto sublinhado.

  • tag strong

    A tag strong serve para deixar o texto em negrito.
    Exemplo:

    É muito importante beber água.

  • tag ol

    A tag ol serve para criar uma lista ordenada.
    Exemplo:

    1. Item 1
    2. Item 2

  • tag ul

    A tag ul serve para criar uma lista não ordenada.
    Exemplo:

    • Morango
    • Banana

  • tag li

    A tag li serve para definir cada item de uma lista, seja ela ordenada ou não
    Exemplo:

    • Item da lista

  • tag a

    A tag a serve para criar um link (âncora).
    Exemplo:

    Ir para o Google

  • tag hr

    A tag hr insere uma linha horizontal (separador).
    Exemplo:

    Parte 1 do texto

    Parte 2 do texto

  • tag sub

    A tag sub deixa o texto em subscrito (abaixo da linha).
    Exemplo:

    Fórmula da água: H2O

  • tag sup

    A tag sup deixa o texto em sobrescrito (acima da linha).
    Exemplo:

    102 = 100

  • tag blockquote

    A tag blockquote pode ser usada para citações longas.
    Exemplo:

    "A vida é 10% o que acontece comigo e 90% como eu reajo a isso."

  • tag font

    A tag font pode ser usada para mudar cor, tamanho e tipo de fonte. Mas, o ideal é usar CSS
    Exemplo:

    Texto normal e texto em vermelho.

  • tag del

    A tag del indica um texto removido/excluído.
    Exemplo:

    Preço: R$ 100,00 R$ 79,90

  • tag abbr

    A tag abbr indica uma abreviação ou sigla, mostrando o significado quando o usuário passa o mouse (atributo title).
    Exemplo:

    HTML é a linguagem padrão da web.

E agora que citamos o atributo title da tag abbr, é importante falar sobre os atributos nas tags e alguns exemplos

Atributos

Os atributos são informações extras que podem ser adicionadas dentro da tag de abertura para configurar o comportamento ou a aparência de um elemento, o valor é colocado entre aspas. Nem toda tag precisa de atributos, mas a maioria aceita. Vejamos alguns atributos:

  • Atributo href

    O atributo href é usado na tag a e define o link de destino.
    Exemplo:

    Ir para o Google

  • Atributo id

    O atributo id identifica de forma única um elemento na página.
    Exemplo:

    tag p com id="intro" para definir que aquele é um parágrafo de introdução.

  • Atributo class

    O atributo class agrupa elementos para aplicar CSS ou JS.
    Exemplo:

    tag p class="texto-destaque" e todos os elementos com essa class poderão receber o mesmo estilo

  • Atributo target

    O atributo target é usado na tag a e define como o link será aberto.
    Exemplo:

    Abrir o Google na mesma aba
    Abrir o Google em nova aba
